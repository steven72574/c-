### 模板
c++另一种编程思想称为泛型编程，主要利用的技术就是模板  
c++提供两种模板机制：函数模板和类模板  
```c++
#include <iostream>
using namespace std;

template <class T>
//template <typename T>
void swapNum(T& a, T& b) {

    T temp = a;
    a = b;
    b = temp;

}
int main()
{
    int a = 10;
    int b = 20;
    //1.模板调用方式
    swapNum(a, b);
    //2.加<>
    swapNum<int>(a, b);
    cout << a << " "<< b << endl;

}
```
